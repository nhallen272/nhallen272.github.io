Raspberry Pi Oyster Float Project

During my internship at VMASC, I worked on a project to embed a Raspberry Pi inside a FlipCage oyster cultivation system to capture live video from a camera and record data from an Inertial Motion Unit (IMU). The IMU measures four types of motion data: angular velocity vector/gyroscope, acceleration, magnetic field strength, and gravitational acceleration. The underwater camera is mounted inside the cage and provides a live video stream of the oysters via YouTube. The code for this project was comprised of about 50/50 Python and bash scripts. The software development leveraged various utilities builtin to the Linux operating system such as creating System-d unit files and cron jobs to ensure the scripts were running at at appropriate times in the background as daemons. Another major aspect of this project was the design of a waterproof physical enclosure for the Raspberry Pi and the sensors. Since it wouldn't be possible to provide any wired power connection, we utilized batteries and solar panels to power the Pi. Using the CAD/3D modeling software, Fusion 360, we designed an enclosure that provides adequate protection from the environment and provides a way to mount the solar panels to power the system. The enclosure was 3D printed using ASA filament for increased resistance to sun damage. Since there was no clear way to mount the solar panels to the float, we designed parts that secured the panels to our enclosure in an innovative way. The GitHub page for the project can be found here: VABlueTech_FlipCam
